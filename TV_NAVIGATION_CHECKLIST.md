# AniVerse TV Navigation Checklist (Controle Remoto)

Use esta checklist para validar rapidamente a navegação no modo TV (`tv-mode`) em cada tela.

## 1) Menu lateral
- [ ] `↑/↓` move entre: Início, Animes, Filmes, OVAs, Coleções, Músicas, Histórico.
- [ ] O item focado fica visualmente destacado.
- [ ] `OK` no item selecionado abre a seção correspondente.
- [ ] Após abrir uma seção, a barra lateral recolhe e o foco vai para o conteúdo.
- [ ] `←` em qualquer card/conteúdo volta para a barra lateral.

## 2) Início
- [ ] Em **Novos Lançamentos**, `→/←` percorre os cards da linha.
- [ ] Em **Novos Lançamentos**, `↓` vai para o primeiro item do **Catálogo Completo**.
- [ ] Em **Catálogo Completo**, `↑/↓` percorre a lista/grade sem salto aleatório.
- [ ] O primeiro card não fica cortado na lateral.

## 3) Animes / Filmes / OVAs
- [ ] Seções exibem layout de grade padrão (igual ao site normal).
- [ ] `↑/↓/←/→` percorre cards sem “teleporte” de foco.
- [ ] `OK` abre o anime selecionado.

## 4) Músicas
- [ ] Cada série aparece com trilha horizontal de músicas.
- [ ] `←/→` percorre músicas da série.
- [ ] `↑/↓` troca entre séries.
- [ ] `OK` reproduz a música focada.

## 5) Modal de vídeo (player)
- [ ] O player abre em tela cheia no modo TV.
- [ ] Fundo principal não interfere/“vaza” enquanto o modal está aberto.
- [ ] `←/→` e `OK` funcionam nos controles do player.
- [ ] Seletores de temporada/episódio recebem foco e podem ser alterados com controle.
- [ ] `Return/Back` fecha o modal ou volta um nível.

## 6) Personalizar Perfil
- [ ] `↓` sai do campo Nome e vai para próximo controle.
- [ ] Não fica preso entre apenas “Nome” e “X”.
- [ ] `OK` permite editar Nome/Senha quando focado no campo.
- [ ] `Return/Back` fecha o modal e retorna ao menu/fluxo correto.

## 7) Indicações visuais
- [ ] Hint de controles aparece no canto inferior direito.
- [ ] O contexto do hint muda conforme foco (Menu / Seção / Catálogo / Perfil / Player).
- [ ] Ícones compactos da barra lateral são flat (sem emoji).
- [ ] Linhas/bordas no modo escuro são claras e no modo claro são escuras.

## 8) Performance básica
- [ ] Navegação por foco sem travadas longas.
- [ ] Vídeo não fica alternando entre carregamento a cada segundo.
- [ ] Quedas de conexão acionam fallback de qualidade automaticamente.

---

## Fluxo rápido de validação (5 minutos)
1. Entrar no site em TV mode.
2. Validar menu lateral (`↑/↓`, `OK`, `←`).
3. Em Início: Novos Lançamentos `↓` para Catálogo.
4. Abrir um anime e validar player/temporada/episódio.
5. Abrir Personalizar Perfil e validar saída do campo Nome com `↓`.
6. Testar seção Músicas e troca entre séries.
